# 🥪 Supersub 

## What are Supersubs?  🥪

Supersubs are superfluid and tradable Subscriptions that unlock more exclusive content the longer they run.


## How to run the DEMO 🏃‍♀️

1. install your dependencies

   ```bash
   yarn install
   ```

2. Generate Contract Types 
   ```bash
   yarn generate
   yarn compile
   ```

3. run the app

   ```bash
   yarn start
   ```
